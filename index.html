Ainda tem Dúvidas?

Respostas diretas para as perguntas mais comuns sobre o nosso serviço.

Como funciona o gerador?

Basta inserir o número de WhatsApp com o DDD, escrever uma mensagem inicial (opcional) e clicar em "Gerar Link". A ferramenta cria um link no formato oficial da API do WhatsApp, pronto para ser compartilhado.

É seguro usar este serviço?

Sim, 100% seguro. A ferramenta apenas utiliza a API pública e oficial do WhatsApp. Nenhuma informação (seu número ou mensagem) é lida, salva ou armazenada. Toda a geração acontece diretamente no seu navegador.

Preciso pagar ou me cadastrar para usar?

Não. O Gerador Whats Brasil é totalmente gratuito, não exige cadastro e não possui recursos pagos ou anúncios. É um projeto de código aberto feito para a comunidade.

Posso usar o QR Code em materiais de marketing?

Com certeza! O QR Code gerado é perfeito para ser usado em flyers, cartões de visita, embalagens, cardápios e qualquer outro material de divulgação para que seus clientes entrem em contato de forma rápida e moderna.
