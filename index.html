Ainda tem Dúvidas?

Respostas diretas para as perguntas mais comuns sobre o nosso serviço.

Como funciona o gerador?

Basta escolher o tipo de gerador (Link ou Contato), preencher as informações, e clicar no botão para gerar seu Link ou QR Code instantaneamente. Para links de WhatsApp, você pode ainda optar por rastrear os cliques.

É seguro usar este serviço?

Sim, 100% seguro. A ferramenta apenas utiliza a API pública e oficial do WhatsApp ou gera dados de contato (vCard) localmente. Nenhuma informação é lida, salva ou armazenada em servidores. Tudo acontece no seu navegador.

Preciso pagar ou me cadastrar para usar?

Não. O Gerador Whats Brasil é totalmente gratuito, não exige cadastro e não possui recursos pagos ou anúncios. É um projeto de código aberto feito para a comunidade.

Posso usar o QR Code em materiais de marketing?

Com certeza! O QR Code gerado é perfeito para ser usado em flyers, cartões de visita, embalagens, cardápios e qualquer outro material de divulgação para que seus clientes entrem em contato ou salvem seu número de forma rápida e moderna.
